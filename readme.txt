Keymail provides a secure, straightforward method for customers to log into your site without passwords only with email 


== Description ==

Keymail login allows to register users only by email, after registration on the user email comes a link for autorisation.

You can add form registration to your blog using sidebar widgets or shortcode ['kml_login_form'].
There is an additional option display all the pages of the blog under the email form. User can choose page on which his will redirect after autorisation.

== Installation ==
1. Upload the `keymail-login` folder to the `/wp-content/plugins/` directory  
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add email form the instructions in the Usage section.
